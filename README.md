## Features  
  
- Lists and selects available microphone (input) devices  
- Real-time volume meter display  
- Lightweight, always-on-top widget  
- Minimal UI for unobtrusive desktop monitoring
- System tray icon that changes colors based on input levels.
- Resize it to your needs
- Remembers chosen device and widget location
